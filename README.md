# Cart Abandonment & Funnel Analysis

## Objective
The goal of this project is to analyze user behavior in an e-commerce platform and identify the reasons behind cart abandonment.

---

## Dataset
- 5,000+ user interaction records
- Includes user actions like view, add to cart, checkout, and purchase

---

## Analysis Performed
- Funnel Analysis (View → Add to Cart → Checkout → Purchase)
- Cart Abandonment Rate Calculation
- Device-wise user analysis
- Category-wise analysis

---

## Key Insights
- ~55% cart abandonment observed between add-to-cart and purchase stages
- Mobile users dominate platform usage
- Consistent drop observed across funnel stages
- User engagement distributed across multiple product categories

---

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab

---

## Conclusion
The analysis highlights key drop-off points in the purchase funnel and suggests opportunities to improve conversion rates through better checkout experience and user interface optimization.
